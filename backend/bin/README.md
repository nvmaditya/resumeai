# Tectonic binary (not committed)

Place the engine here for Overleaf-like PDF output:

```
backend/bin/tectonic.exe   # Windows
backend/bin/tectonic       # Linux/macOS
```

Download: https://github.com/tectonic-typesetting/tectonic/releases

Resolution order: `TECTONIC_PATH` env → this folder → system PATH.
